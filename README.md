# ThriveNext SEO

ThriveNext is India's leading local SEO & GMB optimization agency. This repository contains the front-end code for the main website, featuring a responsive, modern design tailored for maximizing local search conversions.

## Tech Stack
- Vite
- TypeScript
- React + Tailwind CSS
- Framer Motion
- Shadcn/ui

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Build for production:
   ```sh
   npm run build
   ```

## Deployment
Code is optimized and ready for immediate deployment to Vercel or similar platforms.
